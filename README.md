# CLI To-Do Application

A beginner-friendly Command Line Interface (CLI) To-Do application built with Python.

This project helps practice:
- Python fundamentals
- Functions and modules
- File handling
- JSON storage
- Error handling
- Git and GitHub workflow

---

# Features

✅ Add tasks  
✅ View tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Search tasks  
✅ Persistent JSON storage  
✅ Error handling and validation  
✅ Clean modular structure

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/python-cli-todo-app.git
```

---

## Navigate to Project

```bash
cd python-cli-todo-app
```

---

## Run Application

```bash
python main.py
```

or

```bash
python3 main.py
```
